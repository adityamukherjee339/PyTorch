Choosing the right optimizer and learning rate schedule is often the difference between a model that converges and one that diverges.

## Learning Rate Schedulers

A constant learning rate is rarely optimal. Schedulers adjust the learning rate during training.

- `StepLR`: Decays the learning rate by gamma every step_size epochs.
- `ReduceLROnPlateau`: Reduces learning rate when a metric has stopped improving.
- `CosineAnnealingLR`: Decays the learning rate using a cosine annealing schedule.

```python
import torch.optim as optim

optimizer = optim.Adam(model.parameters(), lr=0.01)
scheduler = optim.lr_scheduler.StepLR(optimizer, step_size=30, gamma=0.1)

for epoch in range(100):
    train(...)
    val(...)
    # Step the scheduler AFTER the optimizer step
    scheduler.step()
```
